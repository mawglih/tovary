Написать слово и изменить его.

Сначала слово в единственном числе, потом надо написать его во множественном числе, добавляя числительные «два» и «пять».
  • Один стул, два …, пять …
  • Одна книга, две ..., пять …
  • Одна игрушка, две …, пять …
  • Одно дерево, две ..., пять …
  • Один карандаш, два …, пять …
